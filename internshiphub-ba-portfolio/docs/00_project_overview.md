# 00. Project Overview

## Project Name

InternshipHub – Internship Application and Interview Management System

## Background

Many companies manage internship recruitment through email, spreadsheets, chat tools, and manual notes. This process works when the number of candidates is small, but it becomes difficult to control when applications increase.

## Problem Statement

The current internship recruitment process is fragmented across email, spreadsheets, chat tools, and manual notes. This makes it difficult for HR to track candidate status, schedule interviews, collect interviewer feedback, and provide timely updates to candidates.

As the number of applicants increases, the process becomes harder to manage, less transparent, and more error-prone.

## Business Goals

1. Centralize internship application management in one system.
2. Help HR track candidate status more easily.
3. Improve communication between HR, candidates, interviewers, and hiring managers.
4. Standardize interview feedback.
5. Provide a clear recruitment pipeline for decision-making.

## Project Scope

### In Scope

- Candidate application submission
- Candidate profile and CV upload
- HR candidate screening
- Candidate status management
- Interview scheduling
- Interviewer feedback submission
- Hiring manager pipeline overview
- Candidate status notification

### Out of Scope

- AI-based CV screening
- Online coding test
- Payroll or employee onboarding
- Full implementation source code
- Integration with external HRM systems
